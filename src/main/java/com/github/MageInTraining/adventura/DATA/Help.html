tsMOVE cíl

Přesune hráče do prostoru cíl, který musí být aktuální sousedem prostoru, v němž se hráč právě nachází.

tsUSE předmět

Hráč použije s předmět v místnosti.

tsPICK_UP předmět

Hráč zvedne předmět v místnosti a dá si jí do inventáře.

tsPUT_DOWM předmět

Hráč vyhodí předmět z inventáře.

tsGIVE předmět příjemce

Hráč dá věc z inventáře osobě (příjemce) ve hře.

tsEND

Ukončí hru předběžně.

tsHELP

Příkaz pro vyvolání nápovědy.
